# Travel journal

Travel journal is a personal travel tracking webpage. 

## Installation

Install dependencies

```
npm install
```

Run the project 

```
npm run dev
```

## Current stage
In development

## Usage

In 'My places' you can upload a new city to the visited list or bucketlist.
